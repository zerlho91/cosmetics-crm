# 업그레이드 로드맵 (향후 발전 플랜)

현재 버전은 "기존 데이터 기반 모바일 웹앱 + 기본 AI" 까지 완료된 상태입니다.
아래는 앞으로 탑재하면 좋은 기능들을 **우선순위/난이도** 와 함께 정리한 플랜입니다.
(원하는 항목을 고르면 그 단계만 골라서 작업 가능)

---

## 🟢 Phase 1 — 지금 바로 쓸모 (쉬움 · 추천 먼저)
실사용 안정성과 즉시 체감되는 편의 기능.

| 기능 | 설명 | 난이도 |
|---|---|---|
| **로그인 보안 강화** | 평문 비밀번호 → 해시(bcrypt), 진짜 세션/JWT, `0000` 코드 제거 옵션 | 중 |
| **고객 데이터 보호(RLS)** | Supabase Row Level Security 적용, 실데이터 대비 | 중 |
| **데이터 백업/내보내기** | 전체 고객·매출 엑셀/CSV 다운로드 버튼 | 쉬움 |
| **검색·필터 강화** | 고객/판매 다중 조건 검색, 기간 필터 | 쉬움 |
| **푸시/알림 톤** | 생일·재구매 임박 고객 "오늘의 할 일" 요약 카드 | 쉬움 |
| **앱 아이콘(PNG)** | 홈화면 설치 시 예쁜 전용 아이콘 | 쉬움 |

## 🔵 Phase 2 — 매출을 올리는 기능 (중간)
영업/마케팅에 직접 도움.

| 기능 | 설명 | 난이도 |
|---|---|---|
| **AI 메시지 자동 작성** | 고객별 맞춤 카톡/문자 초안 생성 → 복사·발송 | 중 |
| **문자/카톡 발송 연동** | 알리고·솔라피 등 API로 실제 문자 발송 | 중 |
| **재구매 캠페인 관리** | 대상 고객 그룹 묶어 캠페인 발송·결과 추적 | 중 |
| **상품 추천 엔진** | 구매 패턴 기반 "이 고객엔 이 제품" 추천 정교화 | 중 |
| **매출 목표/대시보드 확장** | 월 목표 대비 달성률, 직원별 실적 | 중 |

## 🟣 Phase 3 — 규모를 키우는 기능 (큰 작업)
여러 매장/직원, 고도화.

| 기능 | 설명 | 난이도 |
|---|---|---|
| **멀티 매장/직원 권한** | 지점별 데이터 분리, 역할별 권한(관리자/직원) | 높음 |
| **예약/방문 관리** | 캘린더 예약, 노쇼 관리, 알림 | 높음 |
| **포인트/멤버십 고도화** | 등급제, 적립/사용 정책, 쿠폰 | 중 |
| **결제 연동** | 실제 결제(PG)·영수증 | 높음 |
| **AI 상담 챗봇** | 고객 응대/제품 문의 자동 응답 | 높음 |
| **분석 리포트 자동발송** | 매주/매월 AI 요약 리포트 자동 생성·전송 | 중 |

---

## 추천 진행 순서
1. **Phase 1 의 "로그인 보안 + 데이터 백업"** — 실고객 데이터를 넣기 전 필수
2. **Phase 2 의 "AI 메시지 작성 + 문자 발송"** — 가장 빠르게 매출 체감
3. 이후 운영하며 필요에 따라 Phase 3 선택

> 작업 요청 예: "로드맵 Phase 1의 데이터 백업 기능 만들어줘" 처럼 항목만 찍어 주면 됩니다.
